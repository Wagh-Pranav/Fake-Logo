How to train, test & build the logo detection dataset?

First, navigate to project's directory using Command Prompt or Terminal.

Then, move on further to "detection" directory by following command:
cd detection

Then, move on further to "files" directory by following command:
cd files

Here you will find the dataset folder named as flickr_logos_27_dataset which contains logo images.

Now move ahead to work with dataset by following steps one by one.

1. python crop_and_aug.py : Crop brand logo images from the flickr_logos_27_dataset and apply data augmentation method. Finally the dataset consists of 217488 images.
2. python gen_train_valid_test.py : Generate(Split) train/valid/test set from the dataset.
3. python train_deep_logo_cnn.py : Train the convolutional neural networks and save the trained model to disk.
4. python test_deep_logo_cnn.py : Test the trained model.

Note: All above process may take upto 1 hour or more.
